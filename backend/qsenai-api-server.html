Users

 

POST Criação de usuário

http://localhost:3333/register

Esse endpoint referente a criação de usuário

Headers
Content-Type
application/json
Body json
{
  "name": "Roberto Amaral da Silva",
  "email": "usuario@email.com",
  "password": "12345",
  "graduation": "Analise e Desenevolvimento de Sistemas"
}
Example request:
Copy to clipboard
curl "http://localhost:3333/register" \
  -H 'Content-Type: application/json' \
  -X POST \
  -d '{
  "name":"Roberto Amaral da Silva",
"email":"usuario@email.com",
  "password":"12345",
  "graduation":"Analise e Desenevolvimento de Sistemas"
}'
Example response - 200:
{
  "newUser": {
    "name": "ABC",
    "email": "ABC@email.com",
    "graduation": "Analise e Desenevolvimento de Sistemas"
  },
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpYXQiOjE1OTQxNjA0ODcsImV4cCI6MTU5NDI0Njg4N30.BezcW5gH1nwkH8ssSJe5_wB9D4wq3SzgcIGhxuCbGvE"
}
Example response - 400:
{
  "error": "Registration failed."
}

http://localhost:3333/answers

Endpoint para enviar resposta a uma pergunta. É enviado no corpo da requisição o id da pergunta e o conteúdo (question_id e content, respectivamente) e o id do usuário que responde é enviado a partir do token no header.

Headers
Content-Type
application/json
Authorization
Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpZCI6MSwiaWF0IjoxNTk0MTU5MTA0LCJleHAiOjE1OTQyNDU1MDR9.uSnTWY_LeFEaYsLRrgK4CnHULjeWv3j7RLjYbVKUo7A
Body json
{
  "content": "No github há a descrição da aplicação e o link correto",
  "question_id": 5
}
Example request:
Copy to clipboard
curl "http://localhost:3333/answers" \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpZCI6MSwiaWF0IjoxNTk0MTU5MTA0LCJleHAiOjE1OTQyNDU1MDR9.uSnTWY_LeFEaYsLRrgK4CnHULjeWv3j7RLjYbVKUo7A' \

  "content":"No github há a descrição da aplicação e o link correto",
